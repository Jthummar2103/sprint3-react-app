# Sprint 3 – Integration Project

This project is developed as part of the Open Source Full Stack Development course.

## Features

* API integration using fetch
* Dynamic routing using React Router
* Display list of users from API
* View individual user details
* Add new user using POST request
* Loading and error handling implemented

## Technologies Used

* React JS
* React Router
* JavaScript (ES6)
* CSS

## How to Run

1. Clone the repository
2. Run `npm install`
3. Run `npm start`

## Deployment

The project is deployed using Vercel and accessible via live URL.

## Notes

* API used is a mock API (JSONPlaceholder)
* Data added using POST request is not stored permanently
